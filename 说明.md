1.将二维小车运动模型修改为三维无人机运动模型，并适配三维的Swarm_lio2

2.将现有的MPC控制算法集成到无人机运动控制上

3.初步调试通过宏观规划算法，现有问题是可能地图数据没匹配上，计算结果和时长有些问题，正在调试...

**终端 1 – Gazebo：**

```bash
ros2 launch gazebo_sim quick.launch.py world:=fishbot2.world pattern:=grid spacing:=2 x:=-3 count:=4
```

**终端 2 – Swarm-LIO2：**

```bash
ros2 launch swarm_lio simulation.launch.py bot_count:=4 rviz_list:=1 use_sim_time:=true
```

**终端 3 – Map fusion：**

```bash
ros2 launch map_fusion map_fusion.launch.py use_sim_time:=true
```

**终端 4 – ESDF map：**
```bash
ros2 launch esdf_map esdf_map.launch.py use_sim_time:=true
```

**终端 5 – Planning：**

```bash
ros2 launch rover3d_navigation planning_apf.launch.py use_gmm_publisher:=true
```

**终端 6 –Controller：**

```bash
ros2 launch mpc_control mpc_control.launch.py robots:=bot1,bot2,bot3,bot4
```
